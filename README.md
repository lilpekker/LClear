# LClear Plugin

## Açıklama
LClear, Minecraft sunucuları için geliştirilmiş bir eklentidir. Sunucu yönetimi ve oyuncu deneyimini iyileştirmek için çeşitli özellikler sunar.

## Özellikler
- Sunucu yönetim araçları
- Oyuncu komutları
- Yapılandırılabilir ayarlar
- Genişletilebilir mimari

## Kurulum
1. Eklenti dosyasını sunucunuzun `plugins` klasörüne atın
2. Sunucuyu yeniden başlatın
3. Gerekli ayarları `config.yml` dosyasından yapın

## Yapılandırma
Eklentinin ayarları `config.yml` dosyasında bulunmaktadır. Bu dosyayı düzenleyerek eklentinin davranışını özelleştirebilirsiniz.

## Komutlar
- `/lclear` - Ana komut
- Diğer komutlar için eklenti içindeki yardım menüsünü kullanın

## İzinler
- `lclear.use` - Eklentiyi kullanma izni
- `lclear.admin` - Yönetici izinleri

## Geliştirme
Bu eklenti Java ile geliştirilmiştir ve Bukkit/Spigot API'sini kullanmaktadır.

## Sürüm
- Versiyon: 1.0.0
- Gerekli Minecraft sürümü: 1.8+
